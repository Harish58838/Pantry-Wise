# PantryWise App

A full-featured real-world smart inventory, shopping checklist, family sync, and AI insights dashboard application!

## Running the App
Since this is a modular architecture using vanilla Javascript, simply load the root folder up in VSCode and start it using a **Live Server** extension, or load `index.html` strictly through an HTTP Server to bypass CORS for local modules (though they are not strict ES6 modules, an HTTP server is recommended).

## Features Include
- 16 Modular CSS Chunks (`/css/`)
- 16 Modular JS Chunks (`/js/`)
- Authenticated Mock OTP Email logic for Reset Passwords
- Mobile ready layout with simulated IoT sensor visualizations.
